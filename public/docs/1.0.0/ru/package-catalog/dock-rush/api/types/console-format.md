# ConsoleFormat

Union-тип форматов console logging.

## Definition

```ts
type ConsoleFormat = 'tree' | 'list' | 'minimal'
```

## Fields

Для union-типа поля отсутствуют.

## Notes

- `tree` — древовидный вывод с отступами.
- `list` — нумерованный список путей.
- `minimal` — краткая сводка по числу директорий и файлов.
