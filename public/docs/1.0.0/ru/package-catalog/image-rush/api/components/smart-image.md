# SmartImage

Backward-compatible alias для [ImageRush](./image-rush.md).

## Назначение

Экспорт указывает на тот же компонент, что и [ImageRush](./image-rush.md).

## Signature

```tsx
const SmartImage: typeof ImageRush;
```

## Parameters

Параметры совпадают с [ImageRushProps](../types/image-rush-props.md).

## Returns

Возвращаемое значение совпадает с [ImageRush](./image-rush.md).

## Usage Notes

В комментарии к исходному коду alias помечен как backward-compatible с указанием на будущее удаление в мажорной версии.

## Example

```tsx
import { SmartImage } from '@da_b1rmuda/image-rush/react';
```
