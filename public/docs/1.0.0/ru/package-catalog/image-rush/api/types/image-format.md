# ImageFormat

Поддерживаемые форматы выходных изображений.

## Definition

```ts
type ImageFormat = 'avif' | 'webp' | 'jpeg' | 'png';
```

## Fields

Полей нет. Это строковый union type.

## Notes

Тип используется в [ImageCandidate](./image-candidate.md), [ImageSourceSet](./image-source-set.md), [ImagePreset](./image-preset.md), [ServerImageRequest](./server-image-request.md) и других публичных контрактах.
