# PlaceholderKind

Поддерживаемые режимы placeholder в preset.

## Definition

```ts
type PlaceholderKind = 'blur' | 'color' | 'none';
```

## Fields

Полей нет. Это строковый union type.

## Notes

Тип используется в [ImagePreset](./image-preset.md), [ImagePresetConfig](./image-preset-config.md) и [PresetOverrides](./preset-overrides.md).
