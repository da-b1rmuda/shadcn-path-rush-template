# CONFIG_RUSH_UPDATE_EVENT

Публичная константа имени Vite HMR event для client config updates.

## Definition

```ts
const CONFIG_RUSH_UPDATE_EVENT = "config-rush:update";
```

## Notes

- Используется [`configRushPlugin`](../functions/config-rush-plugin.md) для targeted config update events.
