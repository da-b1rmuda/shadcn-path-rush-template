# RUSH_MODES

Публичная константа со списком preset modes.

## Definition

```ts
const RUSH_MODES: readonly [
  "development",
  "staging",
  "production",
  "test",
  "preview"
];
```

## Notes

- Используется [`rushModeSchema`](./rush-mode-schema.md).
- Значения совпадают с типом [`RushMode`](../types/rush-mode.md).
