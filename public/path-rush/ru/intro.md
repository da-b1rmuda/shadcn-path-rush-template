# Path Rush 2.0

Path Rush — это file‑based routing движок для Vite + React. Он собирает структуру `pages` в детерминированный манифест и использует лёгкий runtime без доступа к файловой системе. В результате вы получаете стабильный роутинг, лэйауты, loader‑систему и SEO‑генерацию из коробки.

## Что нового в 2.0

- Полная переработка ядра и runtime
- Упрощённый вход через `RouterRoot`
- Строгий `basePath` (маршруты доступны только внутри префикса)
- Настраиваемые `pageFileName` и `layoutFileName`
- Генерация типов маршрутов
- Автоматический `optimizeDeps.exclude` для Vite

## Когда использовать

- SPA на Vite
- Нужен file‑based routing без ручного описания маршрутов
- Важна расширяемость через плагины

## Когда не использовать

- Требуется SSR/SSG прямо сейчас
- Нужны специфичные возможности react-router
