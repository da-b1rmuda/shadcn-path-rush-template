# Поведение 404

Глобальный `not-found.tsx` в корне `pagesDir` используется для всех неизвестных маршрутов.

Для локального 404 разместите `not-found.tsx` рядом с `page.tsx` в нужной директории.

Если `not-found` нет — runtime покажет дефолтную страницу 404.
