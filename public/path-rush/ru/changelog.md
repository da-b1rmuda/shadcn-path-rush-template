# История изменений

## 2.0.0

### Добавлено

- `RouterRoot` для минимального использования
- `pageFileName` и `layoutFileName`
- `generateTypes` для TypeScript
- Иерархическое разрешение `loading.tsx`

### Изменено

- Обновлён публичный API и имя пакета
- `RouterProvider` сам загружает `virtual:routes`
- `basePath` стал строгим
- `interceptLinks` включён по умолчанию

### Миграция

См. **Начало работы → Миграция на 2.0**.
