# Игнорируемые файлы

Если любой сегмент пути начинается с `_`, маршрут полностью исключается.

```
_pages/_internal/page.tsx  # не станет маршрутом
```

Такие файлы можно использовать как вспомогательные и импортировать их в страницах.
